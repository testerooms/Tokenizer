Onboard

Clone, install, configure. The proxy runs as a standalone Node.js service on your infrastructure. Point opencode or Claude Code at localhost:3001 and the whole team is guarded — paid models included.

Requires Node 18+ and an Anthropic API key. 

# clone & install
git clone https://github.com/your-org/tokenizer
cd tokenizer/packages/proxy
npm install && npm run build

# configure & run
export ANTHROPIC_API_KEY="sk-ant-..."
npm start
